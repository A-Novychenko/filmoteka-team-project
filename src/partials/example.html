Зробив деякі налаштування:

  • 1.Підключив шрифт Roboto:wght@400;500
  • 2. Додав favicon
  • 3. Додав у папку images фонові зображення хедера
  • 4.Додав у папку images sprite.svg :
    • href="./images/sprite.svg#icon-film
    • ...#icon-search
    • ...#icon-heart">
    • ...icon-close
    • ...icon-arrow-right
    • ...icon-arrow-left
  • 5. Додав початкові стилі: normalize, common - скидання деяких дефолтних стилів, адаптивний container, базові стилі кнопок (стилі ховеру можна використати для активної кнопки), root - змінні, .visually-hidden
  • 6. Додав базові файли JS: refs - посилання на DOM-елементи, base-config - API-ключ і BASE_URL
  • 7. Додав хедер (хедер динамічно перемальовується, тому під його логіку можна привязати рендеренг головної галереї та рендеринг бібліотеки)
  • 8. Додав футер(в футері ще потрібно зробити модалку "команда")

HTML-partials

The posthtml-include plugin allows you to add code from other HTML files (fragments) to HTML page files. Fragments are stored in the src/partials folder. The addition occurs when compiling page files, such as index.html.

This example.html file is an example of such a snippet, with the markup of a certain part of the main page. This is very convenient for team development on student projects, when several people write in one HTML file at once.

The syntax for adding a fragment to the main file is: <include src="./partials/filename.html"></include>
